## Common Attributes

We've created a list of attributes common to most events. Obviously these will not apply to _all_ events, but it's a good preliminary list to start with.

* Sponsorship
* Safety / Security
* Locations
* Tickets
* Insurance
* Volunteers
* Participating entities
* Marketing
* Set-up Time / Tear-down Time
* License Agreements
* Budgets
* Signage
